# Fansite-Web-App
Fansite Web Application created with React
This project is a Fansite Project about Elliot Page who he is American/Canadian actor.

The project including his movies,tv shows and the movies directed by him in more section in navbar click

In addition there is a "something else" section created with using Google Library API which is located same as movies so in more in navbar click. You can search something and see the pictures and authors. If there is no picture that's mean no picture in array (API)

In home page you have a get touch component which is including name, surname, gender form elements. If you enter all forms and on click the checkbox you will take an alert then see a number in navbar right next to the love icon. The number is a counter that displays fan messages.
